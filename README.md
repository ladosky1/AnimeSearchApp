AnimeLad App - Anime search and watchlist app

This is a fullstack anime discovery platform that allows user to search for anime, view detailed information and manage a personal watchlist.

Built as a learning milestone project to demonstrate real-world frontend and backend integration, state manangement, and modern UI/UX practices.

Tech Stack 

Frontend
* React (vite)
* Tailwind CSS
* React Router
* Context API(Auth + Watchlist)

Backend

* Node.js
* Express
* GraphQL(Anilist API)
* Mongoose

Deployment

* Netlify(Frontend)
* Render(Backend)

Features

* Anime Search with live preview (debounced)
* Detailed Anime page (description, genres, score, trailer)
* Watchlist System(add/remove with instant UI feedback)
* Authentication System(login/ Register/ Logout)
* Optimistic UI updates
* Skeleton Loaders & Loading Statuses
* Toast Notifications
* Fully Responsive Design
* Keyboard Navigation for search

Key Concepts

* FullStack Architecture(Frontend + Backend Separation)
* API integration with GraphQL
* State Management with React Context
* Async Handling and UI feedback
* Component-based UI design
* Tailwind CSS styling
* Error handling and edge cases
